# Project Overview

## What is Storm Tracker?

A tool for tracking, visualizing, and alerting on storm and severe weather events.

## Goals

- Ingest real-time weather and storm data from external sources
- Display storm tracks, intensity, and forecasted paths
- Send alerts for configurable threat thresholds
- Maintain historical storm records

## Out of Scope (initial version)

- Custom forecast modeling
- Mobile native apps
- Social/sharing features

## Status

Initial setup — architecture and conventions are being established.
